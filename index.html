Velkommen til vRops User Group DK

Dette er en user group for folk som benytter vRops og Log Insight eller ønsker at lære mere om disse to produkter. Gruppen er drevet af Michael Monberg som til dagligt arbejder hos VMware som Senior SE og Michael Ryom som arbejder med VMware og automatisering hos Nianet.

Alle er velkommende bare du har lyst til at være med.

 Næste vRops UG er den 26. Juni

Du kan læse mere og tilmelde dig via eventbrite link:

https://www.eventbrite.com/e/vrops-user-group-26-juni-2018-tickets-46689999044

 

Vi kommer til at snakke om hvad der er nyt i vRops 6.7 og snakke om den nye vRO management pack til vRops.

 

Du kan følge os på diverse sociale mediaer eller tilmeld dig vores mailing list og få en mail i indbakken, inden næste vRops User Group møde.
